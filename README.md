# 代码质量检测

一个基于大模型API 的本地桌面工具，用于快速评估代码质量与改进建议。界面为浅蓝底黑字风格，结果以卡片和图表展示，支持多文件项目分析。

## 功能亮点

- 单文件与多文件检测，多个文件会自动视为一个项目
- 支持多语言代码的基础识别如Python、JavaScript/TypeScript、C/C++、Java、Go......
- 维度化评分与建议输出，图表展示整体质量分
- 本地保存配置，API Key 不上传到除接口外的任何地方

### 打开文件

- 点击“打开文件（可多选）”选择代码文件
- 左侧会显示文件列表，点击可预览与编辑
- 右键文件条目可复制文件名或移出检测

### 多文件项目模式

- 当选择多个文件时，会自动视为同一项目
- 分析时会额外关注跨文件逻辑、一致性与架构连贯性

### 开始检测

- 点击“开始检测”
- 右侧展示总体结论、维度图表与具体建议

## 注意事项

- 代码会被发送至你选择的模型接口进行分析，请勿粘贴敏感信息
- 若分析失败，请检查网络、API Key、模型名或 Base URL

## 以下是源码拉取操作 环境要求

- Python 3.10+

## 快速开始

1. 安装依赖

```bash
pip install -r requirements.txt
```

2. 启动应用

```bash
python main.py
```
